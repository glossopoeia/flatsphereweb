{%- comment -%} Unified Content Security Policy for every page on the site. The app needs: - workers (WebGPU shader compilation runs in a worker context) - cdn.jsdelivr.net (Pico CSS, Alpine.js, WESL, image metadata libs) - cloud.umami.is + api-gateway.umami.dev (analytics) - api.allorigins.win (CORS proxy for fetching user-supplied remote images) The standalone pages don't use workers / allorigins, but the looser CSP is harmless for them — no requests will fire to URLs they don't reach. {%- endcomment -%}
