{%- comment -%} Unified Content Security Policy for every page on the site. The app needs: - workers (WebGPU shader compilation runs in a worker context) - cdn.jsdelivr.net (Pico CSS, Alpine.js, WESL, image metadata libs, MathJax) - cloud.umami.is + api-gateway.umami.dev (analytics) - api.allorigins.win (CORS proxy for fetching user-supplied remote images) Blog posts opting into math rendering additionally need: - 'unsafe-inline' in style-src for MathJax's inline style attributes on rendered math elements - font-src https://cdn.jsdelivr.net for MathJax's WOFF font files The standalone pages don't use workers / allorigins, but the looser CSP is harmless for them — no requests will fire to URLs they don't reach. {%- endcomment -%}
