{%- comment -%} Unified Content Security Policy for every page on the site. The app needs: - workers (WebGPU shader compilation runs in a worker context) - cdn.jsdelivr.net (Pico CSS, Alpine.js, WESL, image metadata libs, MathJax) - cloud.umami.is + api-gateway.umami.dev (analytics) - api.allorigins.win (CORS proxy for fetching user-supplied remote images) MathJax (loaded only on posts with `math: true`) additionally needs 'unsafe-inline' in style-src — for the inline style attributes it puts on rendered math — and font-src https://cdn.jsdelivr.net for its WOFF fonts. Both relaxations are gated on page.math, so every other page (the app, the projection gallery, non-math posts) keeps a stricter policy with no inline-style allowance. Nothing else on the site needs them: styling is external CSS plus class toggling, the only programmatic styles are individual CSSOM property writes (which CSP doesn't govern), and there are no web fonts (system font stack). {%- endcomment -%} {%- assign style_src = "'self' https://cdn.jsdelivr.net" -%} {%- assign font_src = "'self'" -%} {%- if page.math -%} {%- assign style_src = "'self' 'unsafe-inline' https://cdn.jsdelivr.net" -%} {%- assign font_src = "'self' https://cdn.jsdelivr.net" -%} {%- endif -%}
